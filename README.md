# EcoSphere

EcoSphere is a modern ESG and sustainability management experience built with React, Vite, Tailwind CSS, and Supabase authentication.

## Prerequisites

1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies with `npm install`.

## Run locally

Start the frontend development server from the project root:

```bash
npm run dev
```

## Build for production

```bash
npm run build
```

## Environment

Create a local environment file with your Supabase settings:

```env
VITE_SUPABASE_URL=your-supabase-url
VITE_SUPABASE_ANON_KEY=your-anon-key
```
